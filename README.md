# 🚨 Click-2-Help

A smart emergency assistance web application designed to provide quick help during critical situations.  
With a single click, users can send alerts, access emergency services, and improve personal safety.

---

## 🌟 Features

- 🚑 One-click emergency alert system
- 📍 Location sharing support
- 📱 Responsive mobile-friendly design
- 🔔 Instant emergency notifications
- ⚡ Fast and lightweight performance
- 🌐 Progressive Web App (PWA) support
- 🔒 Secure environment configuration

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
- Service Workers (PWA)
- JSON
